## My process
1. Created and defined a height for my recipe page card.
2. Used some subjectively appropriate semantic elements to create the text structure.
3. Styled the page with CSS Box Model, CSS Flexbox and Custom Properties.

### Built with

- Semantic HTML5 markup
- CSS custom properties
- CSS Box Model
- Flexbox

### What I learned

1. The marker pseudo-class on list elements,     ::marker 
2. Styling the hr element, particularly using  the background-color instead of color property to set the color; setting the border to none defining a height value.

```css
  hr {
    background-color:hsl(30, 18%, 87%);
    border: none;
    height: 0.01rem;
}

### Continued development

Attempted this challenge with only flex box, box model and CSS custom properties, didn't approach it with other tools like grid, React or Vue becaue they are concepts and frameworks I am yet to master but eventually I will. Gradual but Sure Progress.
